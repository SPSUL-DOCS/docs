# Základy pythonu

## Proměnné

```python title="Deklarovaní proměnné"
<nazev promenne> = <hodnota>

# Příklad
v1 = 22
v2 = 5.5
v3 = "Ahoj"
v4 = True
```
:::warning Boolean
booleany se deklarují s velkým písmenem na začátku `True / False`.
:::

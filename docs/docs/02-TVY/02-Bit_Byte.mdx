# Bit vs. Byte

## Bit
- Základní jednotka informace 
- Obsahuje hodnotu `0` nebo `1`
- Označení `b`


## Byte [Bajt]
- byte= 8bitů
- 
$$
2^8= 256
$$
- Označení `B`


## Další jednotky

- **kB** - Kilobit = 1000`b`
- **KB** - Kilobajt = 1024`B`
- **kiB** - Kibibajt = 1024`B` (moc se nepoužívá)
- **Mb** - Megabit = 1024`KB`
- **MB** - Megabajt = 8`Mb`
- **Gb** - Gigabit = 1024`MB`
- **GB** - Gigabajt = 8`Gb`
- **TB** - Terabajt = 1024`GB`
- **PB** - Petabajt = 1024`TB`
- ...

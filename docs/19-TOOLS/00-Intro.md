# Základní tools

Zde jsou některé základní tools, které budete potřebovat pro vývoj.

## [GIT a GITHUB](./git)

- GIT je **verzovací systém** -> verze programu, ke kterým se dá v případě potřeby vracet
- GITHUB -> stránka, na které se dají verze uložit

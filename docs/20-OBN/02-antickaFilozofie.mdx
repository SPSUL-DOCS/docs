# Antická filozofie

- **Antika** - staré Řecko a Řím.
- Základ veškerého evropského filozofického myšlení.
- Zahrnovala veškeré tehdejší lidské vedení (teprve později se z ní vyčlenily vědní obory).
- Úzký vztah k tehdejšímu náboženství.
